# Contributing to Stratum-TS

Thank you for your interest in Stratum-TS!

## Development Process

1. Fork this repository
2. Create a feature branch: `git checkout -b feature/xxx`
3. Commit your code: `git commit -m ‘feat: xxx’`
4. Push the branch: `git push origin feature/xxx`
5. Submit a Pull Request

## Code Standards

- Use TypeScript
- Maintain ESLint compliance
- New features must include documentation

## Commit Message Guidelines (Recommended)

- feat: New feature
- fix: Bug fix
- docs: Documentation update
- refactor: Code refactoring
